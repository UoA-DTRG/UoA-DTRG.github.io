# Kuaka

{: .note }
> **Coming Soon**: Detailed specifications and documentation for the Bluebird platform.

## Overview

Kuaka is a fixed-wing UAV based in the Aerodynamics Laboratory for outdoors flight research.

## Specifications

- **Type**: Standard Plane
- **Frame**: Volantex Ranger 2400
- **Configuration**: Fixed-wing monoplane, conventional tail, single pusher motor.
- **Actuation**: Single pusher motor thrust, aileron roll, elevator pitch, rudder yaw. Flaps/airbrakes.
