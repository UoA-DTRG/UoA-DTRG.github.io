# Flamingo

{: .note }
> **Coming Soon**: Detailed specifications and documentation for the Flamingo platform.

## Overview

Flamingo is a fully actuated stacked octocopter used in DTRG research.

## Specifications

- **Type**: Fully Actuated Stacked Octocopter
- **Configuration**: 2 layers of 4 rotors each
- **Actuation**: Fixed-tilt rotors providing both vertical and horizontal thrust
