## Getting Started with PX4

> ## Need some more basics first?
> Check out our [Getting Started with DTRG](../guides/Getting-Started) guide.

PX4 is an expansive firmware and implements several useful systems that work together. To start understanding PX4, it's important to familiarize yourself with its architecture and core components. The follow are is a list of a few key concepts to get you started:

### Core PX4 Concepts

- **[PX4 Architecture Overview](https://docs.px4.io/main/en/concept/architecture.html)** - Understanding the modular architecture and system components
- **[Modules and Commands](https://docs.px4.io/main/en/modules/)** - Complete reference for PX4 modules and their functionality
- **[uORB Messaging](https://docs.px4.io/main/en/middleware/uorb.html)** - Inter-module communication system for data exchange
- **[MAVLink Protocol](https://docs.px4.io/main/en/middleware/mavlink.html)** - Communication protocol for ground control and companion computers
- **[Flight Stack Overview](https://docs.px4.io/main/en/concept/flight_stack.html)** - Core flight control algorithms and state estimation
- **[Logging and Analysis](https://docs.px4.io/main/en/dev_log/logging.html)** - System logging, data recording, and flight analysis tools
- **[Parameter System](https://docs.px4.io/main/en/advanced_config/parameters.html)** - Configuration management and tuning parameters
