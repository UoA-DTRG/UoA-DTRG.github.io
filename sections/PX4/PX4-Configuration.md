# Configuration

Configuration of PX4 is primarily done through the QGroundControl (QGC) interface, which provides a user-friendly way to set up and manage your vehicle.

Installing QGroundControl can be done by following the instructions on the [QGroundControl website](http://qgroundcontrol.com/).


## Calibration Process
1. Compass calibration
2. Accelerometer calibration
3. Gyroscope calibration
4. Airspeed sensor calibration (fixed-wing)
5. Radio calibration
